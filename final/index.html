How to use HIP.md

Use this page to manually generate a finalized HIP.md. For the agent-led path, download the bootstrap file, place it at a repo root, and ask the agent to follow it.

  • Manual path: answer the calibration questions, then download the generated HIP.md.
  • Agent-led path: drop the bootstrap HIP.md into a repo root. The agent should create a localhost intake, follow existing UI signals, record your choices, inspect the repo, and replace the bootstrap with the final file.
  • Keep the finalized file alongside AGENTS.md, CLAUDE.md, or your custom agent guidance files.

Use this built-in popup whenever you need onboarding guidance. Repo-local calibration gives the best results because the final file can reflect both your choices and the project context.
